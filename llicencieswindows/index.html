    Llicències Windows

    Aquesta guia ofereix una visió clara i resumida sobre les llicències necessàries per utilitzar Windows Server en entorns professionals. Inclou els tipus de llicències, els requisits per a connexions remotes i la compatibilitat amb diferents versions de Windows als dispositius clients.

    1. Llicència de Windows Server

    Per instal·lar i utilitzar Windows Server en un equip (ja sigui físic o virtual), cal disposar d’una llicència de servidor. Aquesta llicència autoritza únicament la instal·lació i l’ús del sistema operatiu al servidor, però no cobreix l'accés dels usuaris o dispositius. Per això, s’han de gestionar llicències d'accés de manera separada.

    2. Llicències d'Accés al Client (CALs)

    Què és una CAL?

    Una CAL (Client Access License) és el permís legal perquè un usuari o dispositiu pugui connectar-se als serveis del servidor.

    Tipus de CAL

    • CAL per Usuari
      Assignada a una persona concreta. Aquesta pot accedir al servidor des de qualsevol dispositiu.
      Ideal per entorns on cada usuari utilitza diversos dispositius (PC, portàtil, mòbil...)

    • CAL per Dispositiu
      Assignada a un equip específic. Qualsevol usuari pot utilitzar aquest dispositiu per accedir al servidor.
      Molt útil en entorns amb equips compartits o torns rotatius de feina

    3. Connexions Remotes (RDS)

    Quan es fa ús de connexions d’Escriptori Remot (Remote Desktop / RDP), calen llicències addicionals: les CAL RDS (Remote Desktop Services).

    Què necessites per connexions remotes?
    Per a cada connexió remota cal: - Una CAL de Windows Server (per usuari o dispositiu) - Una CAL RDS (també per usuari o dispositiu)

    Aquesta combinació garanteix un accés remot legal i correctament llicenciat.

    4. Compatibilitat del sistema operatiu dels dispositius clients

    Els dispositius que es connectin al servidor han de fer-ho amb una versió de Windows adequada. Aquí tens una comparativa:

    Sistema Operatiu Característiques Adequat per servidor
    Windows Home Ús domèstic, no es pot unir a un domini, no suporta connexions RDP ❌
    Windows Pro Suport per dominis, connexió RDP, gestió empresarial bàsica ✅
    Windows Enterprise Funcions avançades de seguretat, gestió a gran escala, suport per VDI ✅
    Windows Education Basat en Enterprise, amb opcions per entorns educatius ✅

    📝 Encara que utilitzis Windows Pro o superior, encara necessites una CAL per accedir legalment al servidor.

    5. Configuració del servidor de llicències RDS

    Per controlar l'assignació de CALs RDS, s'ha de configurar el servidor de llicències RDS. Aquest s’encarrega de validar i distribuir les llicències durant les connexions remotes.

    Instal·lació del rol

    1. Inicia sessió com a administrador.
    2. Obre l’Administrador del servidor.
    3. Afegeix el rol de Serveis d’Escriptori Remot.
    4. Inclou la funcionalitat de Gestió de llicències RDS.
    5. Estableix l’àmbit com “Aquest domini”.
    6. Finalitza amb “Instal·lar”.

    Activació del servidor de llicències

    1. Obre l’Administrador de llicències RDS.
    2. Clic dret sobre el servidor > Activar servidor.
    3. Segueix l’assistent:
      • Connexió automàtica (recomanada)
      • Introdueix les dades de l’empresa
      • Salta la instal·lació de llicències (es pot fer després)
    4. Finalitza el procés.

    6. Com adquirir les CALs

    Les CALs poden comprar-se: - A través de distribuïdors autoritzats - Directament a la botiga en línia de Microsoft

    Poden adquirir-se individualment o en paquets, segons les necessitats de l'organització.

    7. Resum i recomanacions

    Escenari Tipus de CAL recomanat
    Diversos dispositius per usuari CAL per Usuari
    Ordinadors compartits entre torns CAL per Dispositiu
    Accés remot via RDP CAL de servidor + CAL RDS
    Empresa petita o mitjana Windows Pro o Enterprise
    Ús particular Windows Home (no apte per servidor)

    ✅ Recorda: Complir amb les llicències no només és una qüestió legal, sinó que també garanteix un entorn estable, segur i preparat per créixer.
